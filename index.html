打开终端

![image-20240523105901435](/Users/wuleihuan/Library/Application Support/typora-user-images/image-20240523105901435.png)

下载yolov5并下载相关依赖

git clone https://github.com/ultralytics/yolov5 # clone
cd yolov5
pip install -r requirements.txt # install

(cd yolov5这里注意一下，如果出错的话可以改成yolov5所在的绝对路径)

可以在yolov5下自己创建一个xxx.py文件

python xxx.py运行该文件，可以使用现成的训练好的yolov5s模型进行目标检测。

也可以输入python detect.py --weights yolov5s.pt --source xxxxxx进行检测。

例如python detect.py --weights yolov5s.pt --source 0可以实现摄像头实时目标检测。

![image-20240523110558829](/Users/wuleihuan/Library/Application Support/typora-user-images/image-20240523110558829.png)

或是python detect.py --weights yolov5s.pt --source img.jpg

注意，img.jpg对应的是位于yolov5文件夹下的名称为img.jpg的图像文件

![image-20240523110744874](/Users/wuleihuan/Library/Application Support/typora-user-images/image-20240523110744874.png)

根据输出提示可以去yolov5/runs/detect/exp9下查看检测结果

![image-20240523110842018](/Users/wuleihuan/Library/Application Support/typora-user-images/image-20240523110842018.png)

下一步是YOLOv5部署到web端（flask+js）

省略代码，全部放在yolo_web-main项目里了

app.py代码里有个地方要注意修改一下

这里img_path有可能要改，比如window系统下可能是

img_path = 'runs\\detect\\exp\\' + str(filename)

但macOS是

img_path = 'runs/detect/exp/' + str(filename)

老样子，在终端里进入yolo_web-main目录，然后python app.py，可以根据提示查看本地部署效果。
